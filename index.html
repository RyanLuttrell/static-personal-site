Hi! I'm Ryan

I'm a Full-Stack JavaS cript developer here in the beautiful Calgary, Alberta.

After working for a digital marketing agency for almost a year, I became fascinated with the technology we were using to build responsive websites. I spent some time talking to our Lead Developer to find out more and then started doing some of my own research. After scouring the web for the best way to learn how to code, and making many wrong choices, I came across the Full-Stack JavaScript TechDegree program through Treehouse. What was supposed to take 6-months, only took 3. Every day that I was able to work through the program made me even more excited about the endless possibilities that there are with learning software engineering and drove me to learn more.
